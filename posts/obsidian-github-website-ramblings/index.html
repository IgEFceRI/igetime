Obsidian Github Website Ramblings

i am not a smart person

April 7, 2025   ·   2 min read

#FakeCS #Personal

Somehow, I’ve reached peak jank-ness with this setup of using obsidian to write my blog posts, and then to use github to host a website with all the posts from the markdown files in obsidian.

I do not believe I will ever reach a point when I will call myself an expert in anything and this is proof of that.

My thinking goes that since many people have synced their obsidian to their github, it shouldn’t be impossible to turn that into your website.

Four hours later, I think I was able to prove that:

  1. I’ve truly reached peak jank
  2. I’ve learned powershell against my will
  3. Things may have been easier if I used Linux and command line
  4. Things may have been a lot easier if I just used gitPages
  5. Things may have been a ton more easier if I used tumblr

Oh well, at least I can say I now know how to fanagle multiple different languages and combine it into one monstrous system.

The pipeline works like this:

  1. Everything written piece in obsidian are markdown files
  2. Use Hugo to convert markdown files into html code
  3. That formatting allows you to then push and commit that same code to a github repo
  4. Repo uses the public branch to format a website (don’t forget to include a blank .nojekyll file)

Modifying the site config (hugo.toml) allows you to setup the look of the page, and additional editing in the html files of the theme allow for further customization.

I suppose other than some more customizations, I will be focusing on getting content written first and foremost. It’s been a very very long time since I’ve actually written anything good, and it would be nice to revisit some of that.

I anticipate I will have a lot to say after I receive my typewriter.

Well, that’s all I got for now. In the near future, there will be a more in-detail description of how exactly I got this to work. Alas, it is 4:48 AM in the morning and I do plan to get some sleep tonight.

Farewell and ’til next time!
